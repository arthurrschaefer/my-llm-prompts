# ⚠️ Read This Before Using
Use these prompts at your own risk. While these prompts are designed to be helpful and creative, please keep the following in mind:

* **AI Hallucinations**: LLMs can (and will) occasionally generate incorrect information. Always verify facts, code, or data provided by the AI before relying on it.
* **No Liability**: I provide these prompts "as-is" for educational purposes. I am not responsible for any errors, data loss, or unexpected AI behavior resulting from their use.
* **Token Costs**: Some prompts may be long or trigger long responses. If you are using the API, keep an eye on your usage to avoid unexpected costs.
* **Privacy**: Never paste sensitive personal data or company secrets into a prompt.
